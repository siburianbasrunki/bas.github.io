FrameWork

  • Lavarel

    • Django adalah sebuah kerangka kerja full-stack untuk membuat aplikasi web dengan bahasa pemrograman python. Django juga merupakan kerangka kerja python web tingkat tinggi yang dapat melakukan pengembangan aplikasi dengan cepat dan memiliki desain pragmatis yang bersih.selengkapnya
