Front End Developer

Pendapatan Developer

Berdasarkan informasi dari Neuvoo, rata-rata gaji Front End Developer di Indonesia adalah Rp. 90,000,000 per tahun atau Rp. 46,154 per jam. Gaji awal posisi tersebut dimulai di kisaran Rp. 24,000,000 per tahun atau dua juta perbulan. Sementara bagi karyawan berpengalaman, gaji berada di kisaran Rp. 153,000,000 per tahun atau Rp. 12,750,000 perbulan. Ini asumsinya di Indonesia lho dan angkanya bisa lebih tinggi jika ia bekerja di perusahaan start up terkenal

Teknologi

Front end developer adalah pengembang website yang menggunakan baris kode HTML, CSS, dan JavaScript untuk menghasilkan website dengan tampilan yang menarik. Mereka adalah orang-orang yang mengolah desain murni menjadi website yang interaktif dengan pengguna.

FrameWork

  • React

    • React Native adalah salah satu topik hangat di bidang mobile development selama beberapa tahun terakhir. Salah satu alasan mengapa React Native sangat populer karena menawarkan cara untuk mengembangkan mobile app atau aplikasi selular dengan mudah. Membuat sebuah aplikasi selular bukanlah hal mudah karena membutuhkan coding khusus yang pengembangannya memerlukan waktu lama.Selengkapnya/p>

  • Angular

    • AngularJS merupakan salah satu jenis framework dari JavaScript yang menggunkan metode MVC(model View Controller) dan saat ini dikelola oleh Google.selengkapnya
